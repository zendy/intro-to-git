Git Basics

What is git?

Initialise a git folder

						
							git init

Configure git user name and email

						
							git config --global user.name "Zendy Wongso"
							git config --global user.email "zwongso@seek.com.au"

Check the current status of our working tree

						git status

Add a file to working tree

Let's create a new file

									detail: {

Commit our changes to Staging Area

Branching

What is a branch?

Create a new branch

						
							git branch branchName
							git checkout -b branchName

Show all local branches

						
							git branch

Switch branch

						
							git switch branchName

Merging branch

						
							git merge branchNameToBeMerged

Delete a branch

						
							git branch -d branchName
